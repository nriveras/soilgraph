# Generate positional distortion for boundary shape

Generate positional distortion for boundary shape

## Usage

``` r
generate_boundary_distortion(x_vals, shape = "smooth", seed = 1)
```

## Arguments

- x_vals:

  Numeric vector of x-coordinates

- shape:

  Character: boundary shape type

- seed:

  Random seed

## Value

Numeric vector of y-distortions
